mpvExtended

Advanced mpv-based video player for Android

mpvExtended is a fork of mpv-android, built on the libmpv library. It combines powerful features with an easy-to-use Material3 interface, background playback, and picture-in-picture support. Completely free, open source, and ad-free.

Download APK Star on GitHub

Features

Material3 Design

Modern, expressive UI that follows the latest Android design guidelines.

Advanced Config

Full control over mpv configuration and scripting capabilities.

Picture-in-Picture

Watch your videos while multitasking with seamless PiP support.

Background Playback

Listen to your videos in the background with audio-only mode.

File Management

Built-in media picker with tree and folder view modes.

Subtitle Support

Extensive support for external subtitles and audio tracks.

Network Streaming

Stream videos directly from the network with high-quality rendering.

Gestures

Intuitive zoom gestures and frame-by-frame navigation.

Ad-Free

Completely free and open source without any ads or excessive permissions.
